# MARKETING MODE (Режим маркетолога)

MARKETING MODE предназначен для автоматизации маркетинговых исследований, генерации кампаний, анализа каналов, customer journey, A/B тестов, AI-генерации контента, анализа конкурентов, публикаций, интеграции с BI и генерации презентаций.

## Workflow маркетолога
1. Сбор и анализ рынка, целевой аудитории, трендов (AI-ассистент помогает формулировать гипотезы, анализировать данные).
2. AI-генерация маркетинговых кампаний, customer journey, контента (тексты, баннеры, видео).
3. Анализ каналов: автоматизация сбора, визуализации, публикации метрик (CPC, CPA, ROI, LTV).
4. A/B тесты: AI-генерация гипотез, автоматизация запуска, анализа, публикации результатов.
5. Анализ конкурентов: AI-сбор feature matrix, SWOT, визуализация различий.
6. Автоматизация публикаций: интеграция с соцсетями, корпоративными порталами, рассылками.
7. Интеграция с BI: автоматизация публикации отчётов, дашбордов, инсайтов.
8. Генерация презентаций: AI-генерация презентаций для стейкхолдеров (Google Slides, PDF).
9. Peer review: AI-генерация вопросов для обсуждения, автоматизация сбора обратной связи.
10. Action items: AI-генерация задач, автоматизация публикации в Jira/Confluence.

## AI-генерация кампаний и customer journey
- AI автоматически формирует кампании, customer journey, контент.
- Пример: "AI сгенерировал кампанию для сегмента B2B, customer journey построен."

## Анализ каналов
- Автоматизация сбора, визуализации, публикации метрик (CPC, CPA, ROI, LTV).
- Пример:
```markdown
| Channel   | CPC | CPA | ROI  |
|-----------|-----|-----|------|
| GoogleAds | 1.2 | 10  | 120% |
| Facebook  | 0.9 | 12  | 110% |
```

## A/B тесты
- AI-генерация гипотез, автоматизация запуска, анализа, публикации результатов.
- Пример: "A/B тест: вариант B увеличил конверсию на 8%."

## Анализ конкурентов
- AI-сбор feature matrix, SWOT, визуализация различий.
- Пример: "SWOT-анализ: у конкурента X сильная email-рассылка, у нас — сильнее SEO."

## Автоматизация публикаций
- Интеграция с соцсетями, корпоративными порталами, рассылками.
- Пример: "Пост опубликован в LinkedIn и Telegram."

## Интеграция с BI
- Автоматизация публикации отчётов, дашбордов, инсайтов в Power BI, Tableau.
- Пример: "Отчёт опубликован в Power BI, ссылка добавлена в Confluence."

## Генерация презентаций
- AI-генерация презентаций для стейкхолдеров (Google Slides, PDF).
- Пример: "Презентация для руководства сгенерирована и опубликована."

## Peer review и action items
- AI-генерация вопросов для peer review, автоматизация сбора обратной связи, формирование action items.
- Пример: "Peer review: добавить анализ по TikTok."

## Расширенный чек-лист
- [ ] Маркетинговое исследование проведено
- [ ] Кампании/контент сгенерированы
- [ ] Анализ каналов выполнен
- [ ] A/B тесты проведены
- [ ] Анализ конкурентов проведён
- [ ] Публикации автоматизированы
- [ ] Интеграция с BI реализована
- [ ] Презентации сгенерированы
- [ ] Peer review проведён
- [ ] Action items зафиксированы

## Пример peer review
- "Добавить анализ по TikTok."
- "Пояснить выбор сегмента для кампании."
